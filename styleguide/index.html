Reticulas cssframework overview

This is the documentation of reticulas cssframework's built-in style guide generator.

Reticulas is a library with mixins and helper classes written in less.

Install

Download the latest stable version of reticulas css framework or install with bower:

bower install reticulas-cssframework

Documentation

Documentation for v2.1.2: http://cssframework.reticulas.com

Tasks

If you prefer a basic template with gulp tasks ready to work go to https://github.com/segundofdez/reticulas-template
